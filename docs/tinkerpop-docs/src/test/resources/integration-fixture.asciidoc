= Integration Fixture
:tinkerpop-version: 3.7.7-SNAPSHOT

== Gremlin Block with Modern Graph

[gremlin-groovy,modern]
----
g.V(1)
g.V(1).values('name')
----

== Gremlin Block with Classic Graph

[gremlin-groovy,classic]
----
g.V(1).out()
----

== Bare Gremlin Block

[gremlin-groovy]
----
1+1
----

== Existing Graph Block

[gremlin-groovy,existing]
----
g.V().count()
----

== Gremlin Block with Manual Tabs

[gremlin-groovy,modern]
----
g.V().has('name','marko')
----
[source,java]
----
g.V().has("name","marko")
----
[source,python]
----
g.V().has('name','marko')
----

== Standalone Tab Group

[source,groovy,tab]
----
graph.traversal().V()
----
[source,java,tab]
----
graph.traversal().V();
----
[source,csharp,tab]
----
graph.Traversal().V();
----

== Error Handling

[gremlin-groovy,modern]
----
invalid_syntax_here()
----

== Callouts in Code

[source,java]
----
int x = 1; // <1>
int y = 2; // <2>
----
<1> Sets x
<2> Sets y
